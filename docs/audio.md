## Audio Fields

| Name           | Type            | Optional | Description
| -------------- | --------------- | -------- | --------------------
| thumb          | array           | n        | 内容截图url,[url1,url2]
| original       | string          | n        | 原内容url
| ext            | string          | n        | 后缀名称, 例 'mp3'
| duration       | integer         | n        | 时长, 秒
| size           | integer         | n        | 图片大小, 单位 byte 402269
