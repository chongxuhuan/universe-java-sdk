## Audio Fields

| Name           | Type            | Optional | Description
| -------------- | --------------- | -------- | --------------------
| simple_rate    | integer         | n        | 采样率  (hz)
| bit_rate       | integer         | n        | 码率 （bit）
| ext            | string          | n        | 后缀名称, 例 'mp3'
| duration       | integer         | n        | 时长, 秒
| size           | integer         | n        | 文件大小, 单位 byte 402269
